# AlarmX — Будильник

## Інформація про студента
- **ПІБ:** Говорощук Марія Володимирівна
- **Група:** Кв-33
- **Лабораторна робота №1:** Розробка статичного інтерфейсу Web-додатка
- **Завдання:** Тема — «Будильник»

## Опис
AlarmX — простий web-додаток для керування будильниками. Дозволяє додавати, вмикати/вимикати та видаляти будильники.

## Сторінки
| Файл | Опис |
|---|---|
| `index.html` | Будильники (робоча сторінка) |
| `about.html` | Про додаток |
| `login.html` | Вхід / Реєстрація |
| `profile.html` | Профіль |

## Технології
- HTML5, CSS3
- Bootstrap 5.3
